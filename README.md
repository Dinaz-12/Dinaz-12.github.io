# 🌐 Dinali Chamodya Portfolio

A modern, responsive personal portfolio website built to showcase my projects, technical skills, certifications, and professional journey as a Software Engineering undergraduate.

## 🚀 Live Demo

🔗 https://dinali-portfolio.vercel.app

## ✨ Features

* Responsive and modern user interface
* Dark-themed professional design
* Project showcase with GitHub repository links
* Skills and technologies section
* Certifications gallery
* Downloadable CV
* Contact form with EmailJS integration
* Smooth animations using Framer Motion
* Fast deployment with Vercel

## 🛠 Tech Stack

### Frontend

* React.js
* TypeScript
* TanStack Router
* Tailwind CSS
* Framer Motion

### Services & Deployment

* EmailJS
* Vercel

### Development Tools

* Git
